# MoviePlate

### 背景

这是一个可以查看最新电影，附近影院的 App，
这也是一个用 React-Native 开发的原生态 App，
目的在于将 RN 的使用记录下来，
并且帮助更多的人学会使用 RN。

### 简介

整个项目是用 RN 来创建并且编写的，调用 IOS 原生的部分控件，对 React，React-Native 的学习的一份使用 App。
